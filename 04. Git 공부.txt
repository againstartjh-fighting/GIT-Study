* rm 명령어
- (remove 의 약자인) rm 은 파일을 영구적으로 삭제하게 하는 명령어이다.
- Windows 에서도 똑같은 명령어를 사용한다.

* rm -rf 명령어
- rm 명령어가 파일을 영구적으로 삭제하는 것이라면, rm -rf 는 폴더를 영구적으로 없애는 명령어이다.

* ls - a 명령어
- ls -a 명령어는 hidden file 들도 보여주게 하는 명령어이다.

* git status 명령어
- git status 는 깃의 저장소 정보를 알려주게 한다. 다만, 깃에 저장소가 없으면, '깃 저장소가 아닙니다'
라고 뜬다.
- 또한 찾아보니, 정확하게 말하면, 현재 파일들의 상태를 확인한다고 했다. 커밋이 되었는지, 아직 커밋을
안 한 건지 알려준다고 하였다.

* git init 명령어
- git init 는 git 저장소로 지정하지 않은 폴더를 이제 git 저장소로 지정을 하려고 할 때, 사용한다.
깃의 저장소로 맨 처음 사용할 때, 무조건 적어야 하며, 딱 한번만 초기화를 시키면 된다. 즉, 초기화를 했다면
초기화를 하지 않아도 된다는 말이다.
