* ls 명령어
- ls 는 지금 현재 내가 위차하고 있는 곳에서 모든 폴더들이나 파일들을 보여준다.
- 현재 위치에서 ls (폴더 이름명) 을 입력하면 이동 한 곳에 폴더들을 보여준다.
- windows 에서도 똑같은 명령어이다.

* open (공백). 
- open . 은 맥 전용으로 Finder 을 열게 해주는 기능이다.
- open (폴더 이름명) 을 입력하면, 폴더가 직접 열린다.
- Windows 에서는 start . 을 입력하면 Windows 탐색기가 나온다.

* clear
- 터미널에서 작업한 명령어들을 지우개처럼 깨끗하게 없애준다.
- Windows 에서도 똑같은 명령어와 똑같은 행동을 한다.

* pwd
- 현재 위치를 알려준다.
- Windows 에서도 똑같이 작용하는 것 같다.

* cd
- cd (폴더명) 을 입력하면, 내가 직접 그 폴더 안으로 들어간 상태이다.

* cd (공백)..
- 한 단계 뒤로 가게 해주는 명령어이다.
- Windows 에서도 똑같은 명령어를 가진다.

* touch (파일명)
- touch 명령어는 파일을 만들 수 있다. 예를들어 내가 good.txt 파일을 만든다고 가정하면,
만들고 싶은 위치에 touch good.txt 를 입력해주면 파일이 만들어진다. 
- 여러 개의 파일을 동시에 만들고 싶으면 'touch red.txt good.html god.css 이런식으로 공백을
유지하면서 파일과 확장자명을 동시에 적어주면 된다. 

* mkdir 폴더명 (Make Directory 의 약자이다)
- mkdir 은 폴더를 만들어주는 깃 명령어이다.
